We'll answer those questions in this video.

Edge computing is about facilitating faster, more responsive computing power near the places that need it.

Cloud based applications are powerful and resistant to catastrophic failure, but can be subject to network latency due to greater distance from the source.

Edge computers vary greatly in size and scale.

They can be anything from local mech data centers all the way down to laptops and micro controllers.

Vantiq edge installations are Vantiq platform installations.

They aren't clustered like cloud installations, but they can run offline or in places with spotty networks.

Edge applications generally speaking do some local processing then upload the results to a cloud based installation for further processing and or persistence.
