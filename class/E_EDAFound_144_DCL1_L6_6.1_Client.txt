### Summary

This content provides an overview of the Vantiq Client, focusing on its functionality and how to navigate and configure it. The Vantiq Client is a tool for building low-code visual user interfaces that can run on both desktop browsers and the Vantiq mobile app. It employs the MVC (Model-View-Controller) design strategy, allowing developers to manage data, views, and controllers effectively. The client builder interface includes a palette of widgets that can be dragged onto a canvas and configured. Developers can set client properties, manage data streams, and write custom JavaScript code for event handling. The client properties menu includes tabs for Events, Theme, Localization, and Advanced settings, allowing for customization of appearance, language, and operational characteristics. The content also covers how to add widgets, configure layouts, and test the client.
