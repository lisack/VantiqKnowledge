Prompt Templates Anyone working with AI knows that good outcomes require good prompt engineering.

In this video, we'll see where prompt templates are stored, referenced, and used with runtime configurations in generative AI procedures.

In Vantiq, developers store AI prompts as documents that are then referenced from Gen AI procedures later.

To create a document, just click the + to type a new one or upload one from your file system.

Documents are also not just text, but a wide variety of textual, image, and even video formats.

Make sure the type reflects what kind of format it is.

Creating most resources in the IDE automatically makes them part of the current project, but this is not true of documents.

Should you later decide to export your project and use it elsewhere, the documents that haven't been specifically added to the project will not be in the export file.

Here's the prompt we're using for our Quiz Maker application.

Developers won't always know what the full prompt will be depending on user input or other conditions.

In such cases, create variables that can be substituted in at runtime.

These are delineated as names bracketed by curly braces starting with a dollar sign.

Prompt Templates can be used in multiple places in the platform.

In Gen AI Builder procedures, use the prompt from Template Component to reference them as template documents.

You won't have to type in the name of your template, just select it from the drop down.
