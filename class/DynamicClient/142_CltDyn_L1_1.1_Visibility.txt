**Summary:**

This video tutorial, titled "Widget visibility and appearance," demonstrates how to dynamically control the visibility and appearance of widgets on a Vantiq client page to meet runtime needs. The example used is a visitor check-in screen where widgets appear based on user interaction, such as selecting a status from a drop list. The video explains how a single line of code can make a layout widget and its children visible, and discusses how to change widget properties like background color and transparency using CSS and third-party libraries. It also covers how to dynamically set the stacking order of widgets using specific commands.
