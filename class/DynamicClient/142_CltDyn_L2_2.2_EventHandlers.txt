**Summary:**  
This video tutorial demonstrates how to create dynamic event handlers in Vantiq, both using the client dot add event handler method and through direct coding. It explains the differences between coding event handlers in the Properties pane and using the add event handler method, including how to pass additional parameters like event, page, and extra data. The video also covers creating event handlers for actions not supported by the add event handler method using jQuery or other libraries.
