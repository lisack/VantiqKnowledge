The CLI data manipulation language.

The Vantiq CLI can select, update, and delete persistent data in types.

The CLI can manipulate standard type data and so can the API.

So why would we choose one tool over the other?

Bear in mind that the CLI is meant for doing maintenance tasks on existing resources like bulk uploads, while the API can create and alter those resources.

Generally, inserts and upserts are fine for up to about 1000 records, but the checked versions of them can easily handle even more.

For greater precision in the selection, create a JSON file with the specifications and then use the minus qual option to include that file.

The delete command is a bulk delete.
